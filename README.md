# 🎮 TETRIS - BOOK i LAND

BOOK i LAND が Claude Code と一緒に作ったテトリスゲームです。

▶️ **[今すぐプレイ](https://book-i-land.github.io/Tetris/tetris.html)**

---

## 遊び方

### PC
| キー | 操作 |
|------|------|
| ← → | 左右移動 |
| ↑ | 回転 |
| ↓ | ソフトドロップ |
| Space | ハードドロップ |
| P | ポーズ |

### スマホ
画面下のボタンでタップ操作できます。

---

## 機能

- レベルアップ（10ライン消去ごとにスピードアップ）
- ベストスコア保存
- サウンドエフェクト（🔊ボタンでオン/オフ）
- ネオンフラッシュエフェクト
- PC・スマホ両対応

---

## 技術

HTML / CSS / JavaScript の1ファイルで完結。  
外部ライブラリなし。サーバー不要。
